# J. Synapses
## Summary
In this section of the course, I implemented full closed loop control which followed the map below:

<img src="https://i.imgur.com/w9pXmYl.png" width=90% height=90%>

To do this, I generated synapses connecting the motor and sensor neurons. I simulated them by propagating information from the sensor neurons to the hidden neurons and updating all instances of both neuron types on each simulation time.
